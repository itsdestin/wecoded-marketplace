# DestinCode Themes

Community theme registry for [DestinCode](https://github.com/itsdestin/destinclaude).

Browse and install themes from within the app via **Settings > Themes > Browse Theme Marketplace**.

## Themes

| Theme | Author | Mode | Features |
|-------|--------|------|----------|
| Golden Sunbreak | itsdestin | Dark | wallpaper, particles, glassmorphism, custom-icons, mascot |
| Kitty Pink Classic | claude | Light | pattern |
